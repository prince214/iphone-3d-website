## Run Locally

Clone the project

```bash
  git clone https://github.com/prince214/iphone-3d-website
```

Go to the project directory

```bash
  cd iphone-3d-website
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
